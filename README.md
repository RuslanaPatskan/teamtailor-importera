# Teamtailor Importera

Chrome-розширення для імпорту кандидатів з українських та казахстанських job-бордів напряму в Teamtailor ATS одним кліком.

## Підтримувані платформи

| Платформа | Сторінки |
|-----------|----------|
| **robota.ua** | Відгуки на вакансії, база резюме, профіль кандидата |
| **work.ua** | Відгуки на вакансії, пошук резюме |
| **hh.kz** | Відгуки на вакансії, пошук резюме, профіль кандидата |

## Можливості

- Зелений / червоний / помаранчевий dot-індикатор на кожній картці — одразу видно, чи є кандидат вже в Teamtailor
- Імпорт одним кліком: ім'я, прізвище, телефон, email, фото, посилання на профіль
- Автоматична перевірка дублів перед додаванням
- Масовий імпорт кількох кандидатів одночасно (robota.ua)
- Вибір вакансії, локації, відділу та ролі при імпорті
- Завантаження PDF резюме в профіль кандидата (robota.ua)

## Встановлення

Розширення опубліковано в Chrome Web Store:

**[Встановити Teamtailor Importera](https://chromewebstore.google.com/detail/jkniaddjgpffdkpppgiejcghbfhacndf)**

Після встановлення натисніть на іконку розширення → **⚙️ Налаштування**

## Налаштування

| Поле | Опис |
|------|------|
| **API Ключ Teamtailor** | Отримайте в налаштуваннях Teamtailor → Integrations → API key |
| **Мій тег** | Тег рекрутера, який автоматично додається до кандидата |
| **Вакансія за замовчуванням** | Попередньо вибрана вакансія в модалі імпорту |
| **Локація / Відділ / Роль** | Стандартні значення для нових кандидатів |

## Як користуватись

1. Відкрийте будь-яку сторінку з кандидатами на підтримуваному сайті
2. На картках з'являться кольорові dot-індикатори:
   - 🟢 **Зелений** — кандидата немає в Teamtailor
   - 🔴 **Червоний** — кандидат вже є в базі
   - 🟠 **Помаранчевий** — можливий збіг
3. Натисніть **+TT** на потрібній картці
4. Заповніть або перевірте дані у вікні імпорту
5. Натисніть **Імпортувати**
